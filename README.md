# Hover Tab List

Chrome の縦型タブを **閉じた状態** のまま、画面左端へのマウスホバー（またはキーボードショートカット）で**タブ一覧をふわっと表示**できる Chrome 拡張機能です。

> 縦型タブを開きっぱなしにすると画面占有率が増える。でも毎回ボタンを押して開閉するのは面倒。  
> → 画面左端をかすめるだけで、必要な時だけタブ一覧を呼び出せます。

## 特徴

- **左端ホットゾーン**: 画面左端の約 6px 幅・縦中央 60% の領域にマウスを近づけるとタブ一覧が表示されます。
- **キーボードショートカット**: 任意のキー (推奨: `Cmd+D`) でトグル表示。
- **タブグループ対応**: グループ名・カラー・折りたたみ状態を反映。グループヘッダのクリックで Chrome 側のグループも同時に開閉します。
- **Shadow DOM で完全分離**: サイト側 CSS / DOM とは隔離されており、ページのレイアウトを壊しません。
- **フルスクリーン中は自動無効化**: 動画やゲーム視聴中は誤発動しません。
- **iframe 内には注入しない**: トップフレームのみで動作します。
- **軽量**: ビルドツール不使用、純粋な Vanilla JS。

## インストール (開発者モード)

1. このリポジトリをクローン
   ```bash
   git clone https://github.com/Qota6/chrome_ext_open-close.git
   ```
2. Chrome で `chrome://extensions/` を開く
3. 右上の「**デベロッパーモード**」をオン
4. 「**パッケージ化されていない拡張機能を読み込む**」をクリックし、クローンしたフォルダを選択

## キーボードショートカットの設定

`Cmd+D` は Chrome のデフォルトで「ブックマークに追加」に予約されているため、`manifest.json` の初期値としては登録できません。**手動で設定する必要があります。**

1. Chrome で `chrome://extensions/shortcuts` を開く
2. 「Hover Tab List」の項目を探す
3. 「タブ一覧の表示/非表示を切り替え」の右側の入力欄をクリック
4. 設定したいキーを押す（例: `Cmd+D`）

> `Cmd+D` を割り当てた場合、Chrome 標準のブックマーク追加機能はこの拡張機能で上書きされます。

## 使い方

- **マウスで**: 画面の左端、縦中央あたりにマウスカーソルを近づけると、ふわっとタブ一覧が現れます。
- **キーボードで**: 設定したショートカットキーで表示/非表示をトグル。
- **拡張アイコンクリック**: ツールバーの拡張機能アイコンをクリックでもトグルできます。
- タブをクリックで切り替え、項目右の `×` でタブを閉じることができます。

## 動作しないページ

下記のページではブラウザの仕様上、拡張機能の content script を注入できないため動作しません:

- `chrome://` で始まるページ（設定画面など）
- Chrome ウェブストア (`https://chromewebstore.google.com/`)
- ローカルファイル (`file://`) — 拡張機能の詳細から「ファイル URL へのアクセス」を許可すれば動作します
- 起動直後の新規タブページ (一部)

## ファイル構成

```
.
├── manifest.json   # MV3 マニフェスト
├── background.js   # service worker (ショートカット受信 / タブ API ブリッジ)
├── content.js      # 各ページに注入される UI 本体 (Shadow DOM)
└── README.md
```

## カスタマイズ

`content.js` 冒頭の定数を変更することで挙動を調整できます:

| 定数 | 説明 | デフォルト |
|---|---|---|
| `HOTZONE_WIDTH` | 画面左端の検出幅 (px) | `6` |
| `HOTZONE_TOP` | ホットゾーンの開始位置（上から） | `'20%'` |
| `HOTZONE_HEIGHT` | ホットゾーンの高さ | `'60%'` |
| `SHOW_DELAY` | ホバーから表示までの遅延 (ms) | `120` |
| `HIDE_DELAY` | カーソルが離れてから隠れるまでの遅延 (ms) | `220` |
